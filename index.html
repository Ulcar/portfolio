Unity Projects and games

Discord Channel mirror Overlay app Solo project (2021)

The main challenges for this project were to use the c++ windows API to get transparent windows to work not supported by Unity, This project contains two components: A discord bot that reads all messages, And a unity client that receives the messages from the discord bot. The Unity app is a transparent window (Using the win32 API) that overlays every message the bot receives on the screen. The program also supports Text, Uploaded images, image links, video uploads, and audio uploads and links.

Rootie Tootie Gun Shootie School project (2018)

My first game ever made. It's a top down bullet hell shooter roguelike. (I'm sure you've never heard this concept before). The biggest design challenge here is creating a unique mechanic not present in other of the same genre, so the main gimmick I designed for this game is that the player can also shoot bullet hell, and can use it to block the enemy's bullet hell. This was a duo project, where my responsibilities were programming and designing the player mechanics (Moving, shooting, bullet hell), creating bullet hell patterns using math, Designing and programming enemy AI behaviour, and creating the level design of individual rooms.
